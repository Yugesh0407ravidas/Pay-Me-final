💸 PayNow – Full Stack Payment App

A modern digital payment application built using the MERN stack, enabling users to send, receive, and manage money securely with real-time transaction flow and payment gateway integration.

🚀 Features
🔐 JWT-based Authentication
💸 Send & Receive Money
📊 Transaction History
💼 Wallet System
💳 Razorpay Integration (Test Mode)
⚡ Fast & Responsive UI
🛠️ Tech Stack

Frontend: React (Vite), Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB Atlas
Payment: Razorpay

⚙️ Setup Instructions
1. Clone Repo
git clone https://github.com/your-username/pay-now.git
cd pay-now
2. Backend Setup
cd backend
npm install

Create .env:

PORT=3000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret

Run:

npm start
3. Frontend Setup
cd frontend
npm install
npm run dev
